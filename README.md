# Halı Yıkama Takip Sistemi

Bu C# Windows Forms projesi, bir halı yıkama fabrikasının müşteri ve halı süreçlerini dijital olarak yönetmek için geliştirilmiştir. İşletme sahiplerinin operasyonel yükünü hafifletmeyi ve takibi kolaylaştırmayı hedefler.
• Yeni müşteri kaydı oluşturma ve bilgilerini saklama.
• Sisteme yeni halı ekleme ve detaylarını kaydetme.
• Yıkama aşamasındaki ve teslim edilen halıları ayrı listeler halinde görüntüleme.
• Halıların yıkama veya teslimat durumlarını anlık olarak güncelleme.
